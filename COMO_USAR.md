## 🎨 COMO CUSTOMIZAR:

Para mudar cores:
1. Abra `assets/css/modern.css`
2. Procure por `:root {`
3. Edite as variáveis CSS:
   ```css
   --primary-color: #6366f1;  /* Mude para sua cor */
   --secondary-color: #8b5cf6;
   --accent-color: #ec4899;
   ```

Para adicionar conteúdo:
1. Abra `index.html`
2. Localize a seção desejada
3. Edite o HTML
