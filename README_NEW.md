# Phelipe Mizerani - Portfólio Moderno

## 🎨 Nova Landing Page - Design Moderno e Responsivo

Bem-vindo à minha nova landing page! Este projeto foi completamente redesenhado com tecnologias modernas, oferecendo uma experiência visual impressionante e totalmente responsiva.

### ✨ Características Principais

- **Design Moderno**: Interface contemporânea com gradientes, animações e efeitos de transição suave
- **Totalmente Responsivo**: Funciona perfeitamente em dispositivos mobile, tablet e desktop
- **Performance Otimizada**: Carregamento rápido e eficiente
- **Acessibilidade**: Construído com práticas acessíveis e semântica HTML5
- **Navegação Suave**: Links de navegação com scroll suave e efeitos visuais

### 📚 Seções

1. **Navigation Bar** - Barra de navegação fixa com links para todas as seções
2. **Hero Section** - Apresentação principal com sua foto, título e CTAs
3. **About** - Informações sobre você, experiência e destaques profissionais
4. **Projects** - Showcases dos seus projetos:
   - MizaMoney (Fintech pessoal)
   - MizaInvest (Plataforma de investimentos)
   - Experiência em Empresas (BNDES, Wella, Kérastase)
5. **Skills** - Grid de tecnologias e competências com ícones
6. **Contact** - Informações de contato e links sociais
7. **Footer** - Rodapé com copyright

### 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica moderna
- **CSS3** - Variáveis CSS, Flexbox, Grid, Gradientes e Animações
- **JavaScript Vanilla** - Interatividade e smooth scroll
- **Font Awesome 6.4.0** - Ícones modernos
- **Responsive Design** - Mobile-first approach

### 🚀 Como Usar

1. **Abrir no navegador**: Simplesmente abra o arquivo `index.html` em qualquer navegador moderno
2. **Servidor local** (recomendado):
   ```bash
   python -m http.server 8000
   # ou
   npx http-server
   ```
3. **Deploy**: Envie os arquivos para seu hosting (GitHub Pages, Vercel, Netlify, etc.)
